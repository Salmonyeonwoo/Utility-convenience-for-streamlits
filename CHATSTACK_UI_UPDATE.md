# Chatstack 스타일 UI 업데이트

## ✅ 완료된 작업

### 1. CSS 스타일 업데이트 (Chatstack 스타일)
**파일**: `config.py`

**변경사항**:
- 다크 사이드바 스타일 추가 (`.dark-sidebar`, `.customer-list-dark`, `.customer-item-dark`)
- Chatstack 스타일 메시지 말풍선 (삼각형 포인터 포함)
- 채팅 입력창 컨테이너 스타일
- 고객 정보 패널 스타일

### 2. 채팅 페이지 레이아웃 변경
**파일**: `app.py`

**변경사항**:
- 사이드바에 고객 목록 표시 (다크 테마)
- 메인 영역: 채팅 화면 (2/3) + 고객 정보 패널 (1/3)
- 고객 선택 시 하이라이트 표시
- 읽지 않은 메시지 배지 표시

### 3. 고객 정보 패널 필드 추가
**요청된 필드 모두 추가**:
- ✅ 고객 ID
- ✅ 고객명
- ✅ 연락처
- ✅ 이메일
- ✅ 계정 생성일 (`account_created`)
- ✅ 마지막 접속일 (`last_login`)
- ✅ 마지막 상담일자 (`last_consultation`)
- ✅ 고객 성향 (`personality`)
- ✅ 고객 성향 요약 (`personality_summary`)
- ✅ 설문 점수 (`survey_score`)
- ✅ 응대 평가 점수 (`service_rating`)

### 4. 채팅 입력창 아이콘 버튼 추가
**추가된 버튼**:
- 👤 고객 정보: 고객 정보 업데이트 (구현 예정)
- 🤖 AI 가이드: AI 응대 가이드 표시 (기능 구현 완료)
- 📋 이력: 상담 이력 (구현 예정)

### 5. 샘플 고객 데이터 생성
**파일**: `data/customers.json`

**6명의 샘플 고객 데이터 생성**:
1. 김민수 (신중형)
2. 이지은 (활발형)
3. 박준호 (가족형)
4. 최수진 (프리미엄형)
5. 정태영 (절약형)
6. 한소영 (자유형)

모든 고객 데이터는 요청된 필드를 포함하고 있습니다.

## 📋 레이아웃 구조

### Chatstack 스타일:
```
┌─────────────────────────────────────────────────────────┐
│ [사이드바: 다크 테마] │ [채팅 화면] │ [고객 정보 패널] │
│ - 네비게이션 아이콘    │ - 메시지    │ - 상세 정보      │
│ - 고객 목록 (다크)    │ - 입력창    │                 │
│ - 상담원 프로필       │ - 아이콘 버튼│                 │
└─────────────────────────────────────────────────────────┘
```

### 사이드바 (다크 테마):
- 네비게이션 아이콘 (홈, 채팅, 전화, 고객 데이터)
- 언어 선택
- 고객 목록 (CUSTOMERS 헤더)
  - 고객명 (선택 시 하이라이트)
  - 읽지 않은 메시지 배지 (빨간 원)
- 상담원 프로필 (온라인 상태)

### 채팅 화면:
- 채팅 메시지 표시 영역
- 입력창 + 전송 버튼
- 아이콘 버튼들 (고객 정보, AI 가이드, 이력)

### 고객 정보 패널:
- 모든 요청된 필드 표시
- Chatstack 스타일 레이블 및 값 형식
